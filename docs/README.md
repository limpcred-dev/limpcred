## Visão Geral

O LimpCred é um sistema de gestão empresarial que permite o gerenciamento de múltiplas empresas (multi-tenant), com recursos de:

- Gestão de Empresas
- Gestão Financeira
- Gestão de Clientes
- Gestão de Processos
- Relatórios e Análises
